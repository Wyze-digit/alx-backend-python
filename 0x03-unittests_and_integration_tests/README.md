# Unit Test for access_nested_map

## Description
This project contains a unit test for the `access_nested_map` function located in the `utils` module.  
The purpose is to verify that the function correctly retrieves values from nested dictionaries based on a given path.

## Requirements
- Python 3.7+
- Ubuntu 18.04 LTS environment
- Follows `pycodestyle` version 2.5
- Each file is executable and ends with a newline
- Includes meaningful documentation for all modules, classes, and methods
- All functions are type-annotated

python3 -m unittest test_utils.py
